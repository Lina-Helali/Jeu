# ⭐ Catch the Stars - Jeu HTML/CSS/JavaScript

Un jeu amusant où vous devez attraper les étoiles qui tombent du ciel!

## 🎮 Comment Jouer

- **Déplacez le panier** 🧺 avec votre souris pour attraper les objets
- **Étoiles ⭐** : +10 points
- **Bombes 💣** : -1 vie (à éviter!)
- **Cœurs ❤️** : +1 vie (bonus!)

## 📊 Caractéristiques

- ✨ Système de score en temps réel
- ❤️ Système de vies
- ⏱️ Chronomètre
- 🎨 Design moderne et coloré
- 📱 Compatible mobile et desktop
- 🎯 Animations fluides

## 🚀 Installation Locale

1. Clonez ce repository
2. Ouvrez `index.html` dans votre navigateur
3. Jouez! 🎮

## 📝 Fichiers du Projet

- `index.html` - Structure du jeu
- `style.css` - Styles et animations
- `script.js` - Logique du jeu
